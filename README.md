## 📌 Hướng Dẫn Sử Dụng

1. **Clone** repository về máy:
   ```bash
   git clone https://github.com/vietk5/academic-essays.git
   ```

2. Truy cập vào thư mục học kỳ tương ứng để tìm tài liệu

3. Các file thường ở định dạng **PDF** hoặc **DOCX**
